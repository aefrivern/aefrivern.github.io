深层神经网络

在反向传播训练 w 和 b 时，我们需要计算 dZ[l]dZ^{[l]}dZ[l]，dW[l]dW^{[l]}dW[l]，db[l]db^{[l]}db[l]，方法是通过链式法则求导，向量化之后可以得到：

dW[l]=1mdZ[l]A[l−1]TdW^{[l]} = \frac{1}{m} dZ^{[l]} A^{[l-1]T} dW[l]=m1​dZ[l]A[l−1]T

db[l]=1m∑dZ[l]db^{[l]} = \frac{1}{m} \sum dZ^{[l]} db[l]=m1​∑dZ[l]

dZ[l−1]=W[l]TdZ[l]∗g[l−1]′(Z[l−1])dZ^{[l-1]} = W^{[l]T} dZ^{[l]} * g^{[l-1]\prime}(Z^{[l-1]}) dZ[l−1]=W[l]TdZ[l]∗g[l−1]′(Z[l−1])

最后一层的 dZ[L]=A[L]−YdZ^{[L]} = A^{[L]} - YdZ[L]=A[L]−Y

其中，g[l]′(Z[l])g^{[l]\prime}(Z^{[l]})g[l]′(Z[l]) 是激活函数的导数，∗*∗ 表示对应元素相乘。

如果我们将所有参数初始化为 0，则每一层的梯度都会相同，这样无法训练神经网络，因此我们需要将参数初始化为随机值，比如 W[l]=np.random.randn(shape)∗0.01W^{[l]} = np.random.randn(shape) * 0.01W[l]=np.random.randn(shape)∗0.01。

第 l 层的前向传播：

  • 输入 a[l−1]a^{[l-1]}a[l−1]
  • 输出 a[l]a^[l]a[l]，缓存 z[l]z^{[l]}z[l]

反向传播：

  • 输入 da[l]da^{[l]}da[l]
  • 输出 da[l−1]da^{[l-1]}da[l−1]，dW[l]dW^{[l]}dW[l]，db[l]db^{[l]}db[l]

如果使用较浅的神经网络，则相比深度网络，想要达到相似的效果需要的隐藏层中的单元指数级增长

神经网络的优化

避免过拟合

l2 正则化：J=1m∑L(y,y^)+λ2m∑∣∣W[l]∣∣2J = \frac{1}{m} \sum L(y, \hat{y}) + \frac{\lambda}{2m} \sum ||W^{[l]}||^2J=m1​∑L(y,y^​)+2mλ​∑∣∣W[l]∣∣2

drop out：在训练过程中，以概率 1-keep_prob 将某一层的激活值置为 0，以概率 keep_prob 保留激活值，然后再除以 keep_prob，这样可以减少过拟合。因为这种做法能够防止对一个特定的特征过度依赖，但是对于有大量数据的情况不是很重要

数据增强：比如对于图像数据，可以对图像进行旋转、缩放、裁剪等操作，增加数据的多样性

早期停止：在验证集上的误差开始增大时停止训练，这样可以防止过拟合

对于深层神经网络，由于 W 会大量相乘，导致出现梯度消失或梯度爆炸的问题

可以随机初始化 W：W[l]=np.random.randn(shape)∗np.sqrt(2n[l−1])W^{[l]} = np.random.randn(shape) * np.sqrt(\frac{2}{n^{[l-1]}})W[l]=np.random.randn(shape)∗np.sqrt(n[l−1]2​)，其中 n 是上一层的神经元个数

利用导数的定义，可以检查梯度的导数是否求解正确，方法为：

dJdθ≈J(θ+ϵ)−J(θ−ϵ)2ϵ\frac{dJ}{d\theta} \approx \frac{J(\theta + \epsilon) - J(\theta - \epsilon)}{2\epsilon} dθdJ​≈2ϵJ(θ+ϵ)−J(θ−ϵ)​

其中 ϵ\epsilonϵ 为一个很小的数，比如 10−710^{-7}10−7，然后检查这个值是否和反向传播求得的梯度值相近

加速训练

如果训练集很大，可以使用小的 mini-batch，这样可以加快训练速度

动量梯度下降：目的是让剧烈的震荡变得缓慢，加快平滑的方向

VdW=βVdW+(1−β)dWV_{dW} = \beta V_{dW} + (1 - \beta) dW VdW​=βVdW​+(1−β)dW

W=W−αVdWW = W - \alpha V_{dW} W=W−αVdW​

其中 β\betaβ 为动量参数，一般取 0.9

RMSprop：除以梯度的平方根，以抑制振荡和加速前进

SdW=βSdW+(1−β)dW2S_{dW} = \beta S_{dW} + (1 - \beta) dW^2 SdW​=βSdW​+(1−β)dW2

W=W−αdWSdW+ϵW = W - \alpha \frac{dW}{\sqrt{S_{dW}} + \epsilon} W=W−αSdW​​+ϵdW​

其中 ϵ\epsilonϵ 为一个很小的数，比如 10−810^{-8}10−8

Adam：结合了动量梯度下降和 RMSprop

VdW=β1VdW+(1−β1)dWV_{dW} = \beta_1 V_{dW} + (1 - \beta_1) dW VdW​=β1​VdW​+(1−β1​)dW

SdW=β2SdW+(1−β2)dW2S_{dW} = \beta_2 S_{dW} + (1 - \beta_2) dW^2 SdW​=β2​SdW​+(1−β2​)dW2

VdWcorrected=VdW1−β1tV_{dW}^{corrected} = \frac{V_{dW}}{1 - \beta_1^t} VdWcorrected​=1−β1t​VdW​​

SdWcorrected=SdW1−β2tS_{dW}^{corrected} = \frac{S_{dW}}{1 - \beta_2^t} SdWcorrected​=1−β2t​SdW​​

W=W−αVdWcorrectedSdWcorrected+ϵW = W - \alpha \frac{V_{dW}^{corrected}}{\sqrt{S_{dW}^{corrected}} + \epsilon} W=W−αSdWcorrected​​+ϵVdWcorrected​​

其中 β1\beta_1β1​ 和 β2\beta_2β2​ 为动量参数，一般取 0.9 和 0.999，ϵ\epsilonϵ 为一个很小的数，比如 10−810^{-8}10−8，ttt 为迭代次数

学习率衰减：随着迭代次数的增加，逐渐减小学习率，一种方法是 α=11+decay rate×epoch numα0\alpha = \frac{1}{1 + \text{decay rate} \times \text{epoch num}} \alpha_0α=1+decay rate×epoch num1​α0​

在高维空间，局部最优解很少，大部分都是鞍点，因此梯度下降算法很难收敛到局部最优解，因此不需要担心局部最优解的问题，只需要担心鞍点的问题

另一个问题在于平原处的梯度很小，导致训练速度慢

超参数调整

最重要的超参数是学习率 α\alphaα，其次是 β\betaβ，mini-batch 大小，隐藏层的层数和隐藏单元的个数，λ\lambdaλ，keep_prob 等

在尝试不同的超参数时，可以使用随机搜索，而不是网格搜索，因为有的超参数对结果影响较大，如果采用网格搜索，可能会得到大量相似的无效结果

当发现某个区域的超参数效果不错时，可以在这个区域进行更加密集的搜索

在为超参数设置范围时，可以使用对数尺度，比如 α=10r\alpha = 10^rα=10r，其中 rrr 为一个随机数

对于指数加权平均的超参数，使用 β=1−10r\beta = 1 - 10^rβ=1−10r

超参数训练有两种实践：

  • babysitting one model：手动调整超参数，观察结果
  • train many models in parallel：并行训练多个模型，然后选择效果最好的模型

批量归一化

对于每一层的输入 Z[l]Z^{[l]}Z[l]，可以进行归一化，然后再进行线性变换和激活函数

实现方法为：

μ=1m∑Z[l]\mu = \frac{1}{m} \sum Z^{[l]} μ=m1​∑Z[l]

σ2=1m∑(Z[l]−μ)2\sigma^2 = \frac{1}{m} \sum (Z^{[l]} - \mu)^2 σ2=m1​∑(Z[l]−μ)2

Znorm[l]=Z[l]−μσ2+ϵZ_{\text{norm}}^{[l]} = \frac{Z^{[l]} - \mu}{\sqrt{\sigma^2 + \epsilon}} Znorm[l]​=σ2+ϵ​Z[l]−μ​

Z~[l]=γZnorm[l]+β\tilde{Z}^{[l]} = \gamma Z_{\text{norm}}^{[l]} + \beta Z~[l]=γZnorm[l]​+β

其中 γ\gammaγ 和 β\betaβ 为可学习的参数，分别用来缩放和平移归一化后的值，ϵ\epsilonϵ 为一个很小的数，比如 10−810^{-8}10−8

原来的参数 b 不需要了，因为在归一化后，可以通过 β\betaβ 来平移，而不需要再加上 b

batch norm 有效的原因是：减少了输入值变化的问题，使这些值更加稳定，从而减少了各层之间的耦合，使每一层能够相对独立地学习

其另一个副作用是实现了正则化，因为在训练过程中，每个 mini-batch 的均值和方差都会有所不同，这样可以增加噪声，从而减少过拟合

在测试时，可以使用移动平均来估计 μ\muμ 和 σ2\sigma^2σ2，然后使用这两个值来进行归一化

ML 策略

正交化

因为可调整的超参数太多了，所以应该使每个超参数的调整对应于一个明确的目标，而不是随意调整

应该选择单一数字评估指标

开发集和测试集应该来自同一分布，要尽可能与预期的未来数据一致

但是指标不是一成不变的，例如会随着应用发现有些指标越来越重要，所以可以考虑加权

人类水平表现

机器学习的精度在超过人类水平后，进一步提高的难度会变大，最终会受到贝叶斯误差的限制

在一定程度上，我们可以使用人类水平表现来估计贝叶斯误差，然后再根据这个误差来估计我们的模型是否有进一步提高的空间

有时机器的表现会超过人类，原因是机器能够处理远远超过人类的数据量

误差分析

通过统计开发集中错误的分类来寻找模型的改进方向，或者向其中添加该类型的错误样本

一般来说，训练集中的随机错误不会对模型产生太大影响，但是如果是系统性的错误，那么就需要修正标注的错误

在开发模型的时候，最后是先快速建立第一个模型，然后迭代

有的时候应用的数据不够多，但是有大量和应用数据相似的数据，两者的分布自然有所不同。在划分数据集时，将应用的数据放到开发集和测试集中，这样更能反映模型在应用中的表现，将多余的应用数据和相似数据一起放到训练集中

对于这种做法，为了更好地计算方差以评估模型，将训练集划分出一个 training-dev set，用来评估模型在训练集上的表现，作为中间过渡

  • 训练集和训练-开发集的不同对应方差
  • 训练-开发集和开发集的不同对应数据的不匹配

对于数据不匹配问题，可以人工合成数据，将理想的数据修改为和应用场景一致

但是这么做要注意人工合成的数据有可能全在真实数据的一个子集，这样会导致模型过拟合

传统的方法会将训练划分为几步，每一步分别提取出人类定义的某一个概念（中间件），而端到端的学习方法会直接从输入到输出，这样可以减少人工定义的概念，但是也会增加数据的需求。因此端到端的学习方法适合于数据量大的情况

卷积神经网络

在图像识别上，如果完全采用之前的全连接网络，参数量会非常大，而且会导致过拟合，因此需要使用卷积神经网络

一个可以检测竖直边缘的卷积核：

[10−110−110−1]\begin{bmatrix} 1 & 0 & -1 \\ 1 & 0 & -1 \\ 1 & 0 & -1 \end{bmatrix} ⎣⎢⎡​111​000​−1−1−1​⎦⎥⎤​

也可以进一步调整每一个维度的权重，例如 Scharr 算子：

[30−3100−1030−3]\begin{bmatrix} 3 & 0 & -3 \\ 10 & 0 & -10 \\ 3 & 0 & -3 \end{bmatrix} ⎣⎢⎡​3103​000​−3−10−3​⎦⎥⎤​

当然，这种手工设计的方法不好，这个卷积核的参数是可以通过反向传播来学习的

由于每次卷积都会减小图像的大小，因此可以通过 padding 来保持图像的大小，一般使用的是 zero padding，即在图像的边缘填充 0

  • valid convolution：不使用 padding，输出的大小为 n−f+1n - f + 1n−f+1
  • same convolution：使用 padding，输出的大小为 nnn

卷积步长同样是一个超参数

总结，对于 n x n 的输入，f x f 的卷积核，s 为步长，p 为 padding，输出的大小为：

⌊n+2p−fs⌋+1\left\lfloor \frac{n + 2p - f}{s} \right\rfloor + 1 ⌊sn+2p−f​⌋+1

还有一直想吐槽的，计算机视觉中的“卷积”实际上是互相关 cross-correlation，而不是数学中的卷积，少了一个翻转的操作

将卷积拓展到多个通道，即多通道卷积，这样可以检测多种特征，比如颜色、形状等

对于多通道卷积，卷积核的大小为 f×f×nc[l−1]f \times f \times n_c^{[l-1]}f×f×nc[l−1]​，其中 nc[l−1]n_c^{[l-1]}nc[l−1]​ 为上一层的通道数，输出的大小为 n×n×nc[l]n \times n \times n_c^{[l]}n×n×nc[l]​，其中 nc[l]n_c^{[l]}nc[l]​ 为当前层的卷积核数

池化层 pooling：用来减小图像的大小，减少参数量，一般有两种方法：

  • max pooling：取池化窗口中的最大值
  • average pooling：取池化窗口中的平均值，不常用

池化层只有超参数 f 和 s，其中 f 为池化窗口的大小，s 为步长，没有参数需要学习

卷积网络的参数较少有两个原因：

  • 参数共享：每个卷积核都是对整个图像进行卷积，因此参数共享
  • 稀疏连接：每个输出值只与输入的一小部分有关

实例研究

三个经典的卷积神经网络：

VGG-16 是大力出奇迹的典型，结构简单，胜在层数较多

残差网络 ResNet：看上去和普通的网络没什么区别，但是在卷积层后面加上跳跃连接 skip connection

优点在于可以训练非常深的网络，因为跳跃连接可以使梯度直接传到前面的层，避免了梯度消失的问题

残差网络有效的原因是：在训练过程中，如果某一层的权重为 0，那么这一层的输出就是 0，这样就相当于没有这一层

1 x 1 的卷积核可以通过压缩通道来减少参数量，同时也可以增加非线性（同样使用到了 ReLU）

Inception 网络：使用多个不同大小的卷积核，然后将结果拼接在一起，这样可以让神经网络自己选择要用哪种卷积核

可以发现，如果直接这么做的话，参数量会非常大，因此可以使用 1 x 1 的卷积核来减少参数量

图像识别的数据相对而言还是比较少的，因此经常会使用手工工程和网络架构来提高性能

一些在实际应用中没什么用，但是跑分很高的方法：

  • ensemble：将多个模型的结果取平均
  • multi-crop：对图像进行多次裁剪，然后取平均
