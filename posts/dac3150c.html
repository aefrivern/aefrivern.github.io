实用软件

发表于2024-12-11|更新于2025-03-26|工具
|总字数:1165|阅读时长:3分钟

这里包含了一些我认为比较实用的软件，包括 Windows 和 Android 平台。

Windows 美化

在我看来，Windows 的桌面是用来欣赏的，而不是用来摆放各种杂物的，因此我的桌面总是空无一物。毕竟谁也不想整天盯着那一堆乱七八糟的杂物，没事可以看一下桌面，闲来换个壁纸，放松一下心情。

TranslucentTB 是一个可以让任务栏透明的工具，简单的软件能够极大地增强 Windows 桌面的美感

Wallpaper Engine 是最著名的动态壁纸软件，是 Steam 上最值得购买的游戏。没错，我真的只是拿来放壁纸的

一般来说都会订阅非常多的壁纸，因此这里面的文件夹分类就非常重要，但是文件夹分类没有云同步，因此需要手动同步：复制应用目录下的 config.json 文件

  • 这个文件中包含了 wallpaper engine 的安装目录，需要根据自己的情况修改
  • 第三行是当前用户的用户名，也要更新
  • 底下还有创意工坊的下载目录，也要修改

耦合度这么高，怪不得没有云同步，不是作者不想做，而是实在做不到啊

文件可以整齐地放在文件夹内，但是软件放在哪里呢？Windows 10 中的磁贴是一个完美的设置，但是微软在 Win11 中取消掉了微软的常规操作了，ExplorerPatcher 就可以将这个功能复现出来。此外，还支持自定义任务栏等。

[shortcuts.vdf](C:\Program Files (x86)\Steam\userdata\1132578176\config\shortcuts.vdf)

看图软件

这里最推荐的看图软件是 XnView MP

  • 免费
  • 启动速度快
  • 包括图片浏览器，功能强大
  • 图片的批处理，重命名，格式转换等功能一应俱全，做到了关于图片的 ALL IN ONE

截图工具

PixPin 是出色的截图、贴图工具

  • 支持长截图
  • 支持截动态图片（替代 screen to gif）

快捷键：

  • 截图：Ctrl + 1
  • 贴图：Ctrl + 2

下载工具

qBittorrent-Enhanced-Edition 是 qBittorrent 的增强版，出色的 BT 下载工具

bt 通常需要添加 tracker 才能下载，可以使用 trackerslist 获取最新的 tracker 列表

另一个下载工具是 imFile，这款软件是 motrix 是一个 fork，因为原版 motrix 已经停止维护。这一软件的优点在于支持 rpc，可以通过 api 进行控制。

在接管浏览器下载的时候，可以使用 Internet Download Manager，这是一个非常强大的下载工具，支持断点续传，下载加速等功能。

这个工具是要收费的，所以这里推荐免费的替代品 Free Download Manager，这个工具也是非常强大的，支持断点续传，下载加速等功能。

办公软件

WPS 删除了所有第三方广告 当然潜台词是保留了自己的第一方广告，相当整洁。支持 Word、Excel、PPT、PDF 等格式。

视频播放器

Windows 端推荐直接使用 K-Lite Codec Pack Full，省去各种繁琐的步骤，其中 Full 的版本包含并配置了：

  • mpc-hc：一个轻量级的播放器
  • LAVFilters：一个 DirectShow 解码器
  • madVR：一个高质量的视频渲染器

对于 Android 端，推荐使用 MX Player，这是一个非常强大的播放器，支持多种格式，支持字幕，支持手势控制等。

压缩软件

7-Zip 是一个开源的压缩软件，支持多种格式，压缩率高，速度快。

音乐

移动端的本地音乐播放器推荐 椒盐音乐，是一款非常简洁、美观、功能强大、开箱即用的开源音乐播放器，作者似乎还是国人。

浏览器

御三家（Chrome, Firefox, Edge）都比较常用，其中最常用的是 edge，同步比较方便，又兼容 Chrome 生态，而且不需要另外下载。

一些 edge://flags 中推荐开启的选项：

  • Windows Scrolling Personality 和 Smooth Scrolling：这两个选项可以让滚动更加流畅
  • Fluent overlay scrollbars：这个选项可以让滚动条更加美观
  • Experimental JavaScript、Experimental Web Platform features、Experimental QUIC protocol：支持一些更好的新特性

漫画

mihon 是一个开源的漫画阅读器，支持多种格式，支持自动翻页，支持自动下载，支持自动识别漫画章节等。

软件本体并没有包含漫画源，可以在 这里 添加源。

杀毒软件

尽管有一些广告，但是 360 是毫无疑问的全世界最强的杀毒软件，至于火绒，只是图一乐罢了。
