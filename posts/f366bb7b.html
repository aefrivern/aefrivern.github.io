对于 Windows 用户，这里推荐 WinLibs，里面同时包含了 GCC 和 Clang，以及一些在开发中常用的工具，如 CMake、GNU Make、Ninja、Nasm、Doxygen、cppcheck 等。

vcpkg

vcpkg 是一个 C++ 库管理工具，可以方便地安装和管理 C++ 库，支持 Windows、Linux 和 macOS

安装 vcpkg：

如果使用的是 Visual Studio，可以直接在 Visual Studio 中安装 vcpkg，默认安装目录应该为：C:\Program Files\Microsoft Visual Studio\2022\Community\VC\vcpkg

将其添加到环境变量 Path 中，然后在终端输入 vcpkg，如果出现帮助信息，则说明安装成功

另外，还有将根路径添加到环境变量 VCPKG_ROOT 中，这样在 CMake 中就可以直接使用 vcpkg 安装的库了

vcpkg 有两种模式：经典模式和清单模式，这里推荐较新的清单模式
